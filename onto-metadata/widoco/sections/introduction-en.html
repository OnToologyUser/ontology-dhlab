Introduction back to ToC

As stated in the abstract, the present vocabulary is temporary and defines just a few elements. So far these elements are used to characterize sources in the Garzoni data-set, represented according to the Garzoni Data Model Ontology.

The namespace of the Archive Metadata Model ontology is http://vocab.dhlab.epfl.ch/archive-metadata#, for which we use the prefix meta. Namespaces of other vocabularies used in this document are reported in the table below.
